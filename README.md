# Student Mental Health Analysis

An exploratory data analysis project examining relationships between student mental health, academic factors, and lifestyle variables.

## Project Overview

This project uses Python to analyze student mental health data and identify patterns related to academic pressure, lifestyle habits, and other student characteristics. The goal was to practice data cleaning, exploratory analysis, visualization, and interpretation using a real-world style dataset.

## Features

* Data cleaning and preparation
* Exploratory data analysis
* Summary statistics
* Visualizations of student mental health patterns
* Interpretation of relationships between variables

## Tools Used

* Python
* Jupyter Notebook
* pandas
* matplotlib

## Skills Demonstrated

* Data cleaning
* Exploratory data analysis
* Statistical thinking
* Data visualization
* Communicating insights from data
* Working with real-world survey-style data

## How to View

Open the notebook file:

```text
student_mental_health_analysis.ipynb
```

The notebook contains the analysis workflow and interpretation from the project.
